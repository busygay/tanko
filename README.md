# Tankou（弹口）- 日语学习塔防游戏

![Tankou Logo](icon.svg)

## 项目简介

Tankou（弹口）是一款创新的日语学习塔防游戏，将知识问答与塔防策略深度融合。玩家通过回答日语问题获取战斗资源，利用这些资源和策略性技能来抵御敌人的进攻。游戏的核心在于平衡**知识储备**、**风险管理**和**战术决策**。

### 游戏特色

- **学习与娱乐结合**：通过游戏化的方式学习JLPT N5-N1日语词汇
- **动态难度系统**：根据玩家表现自动调整游戏难度
- **错题本系统**：智能记录和管理错误答题，提供针对性强化练习
- **多样化题型**：包含选择题、单词重组等多种题型
- **技能系统**：丰富的技能树和触发器系统，增强策略性
- **无限关卡**：动态生成的关卡，提供持续的挑战

## 核心机制

### 答题-战斗循环

游戏的核心循环是答题获取资源，然后用资源进行战斗防御：

1. **答题阶段**：回答日语问题，获取战斗资源
2. **战斗阶段**：使用资源防御敌人进攻
3. **成长阶段**：通过经验值升级，解锁新技能和内容

### 资源系统

游戏构建了一个三位一体、相互关联的资源循环系统：

| 资源名称 | 获取方式 | 主要用途 | 战略价值 |
|---------|---------|---------|---------|
| **能量点 (EP)** | 每答对一题获得 `+1` 点；累计答错两题扣 `1` 点 | 积攒以获取行动点 | 核心的长期战略资源 |
| **子弹 (Bullet)** | 每答对一题获得 `+1` 颗；消耗行动点可一次性装满 | 防御塔进行攻击的基础消耗品 | 直接影响战场的即时火力输出 |
| **行动点 (AP)** | 能量点积满 `10` 点时获得 `+1` 点，上限3点 | 消耗 `1` 点可立即重新装满子弹 | 强大的战略级资源，关键时刻扭转战局 |

### 题库系统

游戏采用JLPT（日本语能力测试）标准分级：

| JLPT级别 | 词汇数量 | 难度等级 | 设计目标 | 当前状态 |
|---------|---------|---------|---------|---------|
| **N5** | 600词 | 入门级 | 基础日常词汇，适合初学者 | ✅ 已完成 |
| **N4** | 400词 | 初级 | 基础语法词汇，构建语言框架 | ✅ 已完成 |
| **N3** | 800词 | 中级 | 进阶词汇，提升表达能力 | ✅ 已完成 |
| **N2** | 1000词 | 中高级 | 复杂词汇，接近母语水平 | 🔄 待开发 |
| **N1** | 1500词 | 高级 | 专业词汇，达到熟练运用 | 🔄 待开发 |

## 技术栈和依赖

- **游戏引擎**：Godot 4.x
- **编程语言**：GDScript
- **目标平台**：Windows（可扩展至其他平台）
- **数据存储**：JSON格式（错题本、游戏进度）
- **字体**：思源黑体（Siyuan-NoKorean-Alternative.woff2）

## 快速开始指南

### 环境要求

- Godot 4.5 或更高版本


### 安装步骤

1. **克隆项目**
   ```bash
   git clone [项目仓库地址]
   cd tankou
   ```

2. **打开项目**
   - 启动 Godot 引擎
   - 点击"导入"按钮
   - 选择项目根目录下的 `project.godot` 文件

3. **运行游戏**
   - 在 Godot 编辑器中点击"运行"按钮（F6）
   - 或导出项目后运行可执行文件

### 游戏操作

- **主菜单**：选择游戏模式和设置
- **答题界面**：点击选项回答问题，使用跳过按钮跳过难题
- **战斗界面**：自动攻击范围内的敌人，使用技能增强战斗力
- **商店界面**：升级时使用天赋点购买技能和升级
