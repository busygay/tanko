
## **BT-7271**
### **说明**
    这是BT-7271的技能说明,作为BT技能系列的另一个分支,他和BT-7270同属爱与机器人系列,为了和系列枪斗术对标所诞生的新技能,
### **核心机制**
- **类型**：召唤技能
- **存在时间**：15秒
- **召唤物**：BT-7271
- **存在数量**:1 max
- **移动速度**:50 px (略高于敌人的最高速度)

### **普通攻击逻辑**
- **攻击伤害**：玩家的50%攻击力
- **攻击模式**：随机寻敌 → 单次雷霆打击 → 施加【雷霆标记】 → 立即切换目标

### **召唤技 - 雷霆脉冲**
- **触发条件**：召唤时场上已存在其他BT-7271
- **执行流程**：检测存在 → 清除旧单位 → 触发脉冲
- **攻击范围**: 以召唤物为中心，以200px为半径的圆形,并发生雷电锁链追中带有雷霆标记的敌人
- **攻击伤害**：玩家攻击力的100%

### **雷霆标记系统**
- **标记效果**：减速30%，雷电伤害+25%，持续5秒
